# Vite + Vue 多页面配置

可通过命令行选择打包多项目。

## 实现的功能

- TypeScript + Pinia 
- 请求封装，可取消重复请求
- 代码规范，ESLint + Prettier + StyleLint

## 如何使用

```
# 安装依赖
$ yarn install

# 运行
$ yarn server:dev

# 打包
$ yarn build:prod
```
